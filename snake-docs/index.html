スネークゲーム

クラシックスネークゲームの制作記録

制作ステップ

Step 1

Canvas上にゲームフィールド構築

描画エリア、背景、グリッドを設定。ゲームの土台となるフィールドをCanvas 2D APIで構築する。

ここで学ぶこと: Canvas 2D APIゲームフィールド設計
Step 2

スネークの移動と成長

方向制御、体の追従、食べたら伸びる仕組み。配列で蛇の体を管理し、キー入力で方向を変える。

ここで学ぶこと: 配列操作（push/shift）キーボード入力処理方向ベクトル
Step 3

衝突判定

壁との衝突、自分の体との衝突。ゲームオーバーの条件を実装して、ゲームとしてのルールを完成させる。

ここで学ぶこと: 座標の比較衝突判定アルゴリズムゲームオーバー条件
Step 4

食べ物の生成とスコア

ランダム配置、スコア加算、スピードアップ。食べるたびに難易度が上がる仕組みを実装する。

ここで学ぶこと: 乱数生成難易度の動的調整ハイスコア管理(localStorage)
Step 5

タッチ操作対応

スワイプで方向変更（モバイル対応）。スマートフォンでも快適にプレイできるようにする。

ここで学ぶこと: touchstart/touchendスワイプ方向の計算モバイル対応

使用技術

  • HTML5 Canvas
  • JavaScript
  • localStorage
