メッセージング連携

メッセージングプラットフォームとの連携による自動通知・自動返信

概要

メッセージングプラットフォームのAPIを活用し、顧客への自動通知、自動返信、管理者からの手動メッセージ送信を実装した。Webhookによるイベント受信から、テンプレートメッセージの送信まで、チャットBot連携の全工程を記録。

制作ステップ

Step 1

Webhook受信

メッセージングプラットフォームからのイベント（メッセージ受信、友だち追加など）をWebhookで受信するエンドポイントを実装した。

ここで学ぶこと: Webhook 署名検証 エンドポイント
Step 2

自動通知

顧客のステータスが変更された際に、メッセージングプラットフォーム経由で自動的にメッセージを送信する機能を実装した。

ここで学ぶこと: APIクライアント テンプレートメッセージ 非同期送信
Step 3

自動返信

顧客からのメッセージに対して、定型応答を自動で返す機能を実装した。よくある質問に即座に回答し、対応コストを削減する。

ここで学ぶこと: 自動返信ロジック キーワードマッチ フォールバック応答
Step 4

管理画面からの手動送信

管理者が管理画面から個別の顧客にメッセージを送信できる機能を実装した。自動返信では対応できないケースに対応する。

ここで学ぶこと: 管理者UI API呼び出し 送信ログ

ふりかえり

  • Webhook + 署名検証で、セキュアなイベント受信を実現
  • テンプレートメッセージにより、一貫性のある顧客コミュニケーションを自動化
  • 自動返信と手動送信の組み合わせで、効率的な顧客対応フローを構築

使用技術

  • Python
  • FastAPI
  • Webhook
  • JavaScript
