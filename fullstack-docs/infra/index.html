インフラ

VPS構築、デプロイ、定期実行、環境変数管理

概要

Webサービス全体を稼働させるためのインフラ環境を構築した。VPSの初期設定、アプリケーションのデプロイ、cronによる定期タスク実行、環境変数の安全な管理まで、運用基盤の全工程を記録。

制作ステップ

Step 1

VPS構築

サーバーの初期設定とアプリケーションのデプロイを行った。SSH接続、systemdによるプロセス管理、リバースプロキシの設定など、本番環境の構築手順を確立。

ここで学ぶこと: VPS SSH systemd
Step 2

cron設定

定期タスクの自動実行をcronで設定した。データ集計、バックアップ、通知処理など、定期的に実行すべき処理をスケジューリング。

ここで学ぶこと: crontab 定期実行 ログ管理
Step 3

環境変数管理

APIキーや設定値を安全に管理する仕組みを構築した。ソースコードにシークレット情報を含めず、環境変数として外部から注入するパターンを採用。

ここで学ぶこと: .env 環境変数 シークレット管理

ふりかえり

  • VPSの初期設定からデプロイまでの一連のフローを確立
  • systemdでプロセス管理を行い、サーバー再起動時の自動復旧を実現
  • 環境変数によるシークレット管理で、セキュリティと運用性を両立

使用技術

  • VPS
  • cron
  • Python
