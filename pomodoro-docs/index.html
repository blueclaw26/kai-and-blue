ポモドーロタイマー

集中タイマーアプリの制作記録

概要

25分作業→5分休憩のポモドーロテクニックを実装したタイマーアプリ。

制作ステップ

Step 1

タイマーの基本ロジック

  • setIntervalで1秒ごとにカウントダウン
  • 「作業中」「休憩中」「停止中」の3状態を管理
  • 作業→休憩の自動切替
ここで学ぶこと: setInterval/clearInterval、状態遷移パターン、時間計算
Step 2

SVGプログレスリング

  • SVGの円（circle）でプログレスリングを描画
  • stroke-dasharrayとstroke-dashoffsetで進捗を表現
  • 作業中はティール、休憩中はアンバーに色変更
ここで学ぶこと: SVGの基本、stroke属性、動的なスタイル変更
Step 3

Start / Pause / Reset操作

  • 3つのボタンで操作
  • Pause→Resumeのトグル
  • Resetで状態を初期化
ここで学ぶこと: イベントハンドリング、状態に応じたUI更新
Step 4

ブラウザ通知

  • Notification APIで権限をリクエスト
  • タイマー終了時にデスクトップ通知を送信
ここで学ぶこと: Notification API、権限管理、ブラウザの機能活用
Step 5

終了音（Web Audio API）

  • Web Audio APIでビープ音を生成（外部ファイル不要）
  • OscillatorNodeで2トーンの短いメロディ
ここで学ぶこと: Web Audio API、音声合成の基本
Step 6

設定のカスタマイズと保存

  • 作業時間・休憩時間をユーザーが変更可能
  • localStorageに設定を保存、次回アクセス時に復元
  • セッションカウンター（完了したポモドーロ数）
  • タブタイトルに残り時間を表示
ここで学ぶこと: localStorage API、フォーム入力の処理、タブタイトルの動的更新

Kaiのレビューで変わったこと

  • このアプリはサブエージェントで一発作成、大きな修正なし

使用技術まとめ

  • HTML
  • CSS
  • SVG
  • JavaScript
  • setInterval
  • Notification API
  • Web Audio API
  • localStorage
