絵文字カードゲームの制作記録

概要

4×4の絵文字カードをめくってペアを見つけるメモリーゲーム。JavaScript入門として制作。

制作ステップ

Step 1

HTMLでゲームボードを構築

16枚のカード要素を配置するグリッド構造と、ゲームのUIを設計する。

  • 16枚のカード要素を配置するグリッド構造
  • スコア表示（手数、マッチ数、タイマー）のUI
  • リスタートボタン
ここで学ぶこと: HTML要素の動的生成UIレイアウト設計
Step 2

CSSでカードめくりアニメーション

CSS 3Dトランスフォームを使って、カードの裏表をめくる演出を実装。

  • CSS 3D transform（rotateY）でカードの裏表を表現
  • perspective、backface-visibilityプロパティで奥行きと裏面の非表示を制御
  • マッチ時の緑グロー演出
ここで学ぶこと: CSS 3Dトランスフォームbackface-visibility視覚フィードバック
Step 3

JavaScriptでゲームロジック

カードのシャッフル、めくり判定、タイマーなどのゲームロジックをJavaScriptで実装。

  • Fisher-Yatesシャッフルアルゴリズムでカードをランダム配置
  • 2枚めくり→比較→一致/不一致の状態管理
  • 800msのディレイで不一致カードを裏返す（setTimeout）
  • タイマー — 最初のクリックで開始
ここで学ぶこと: Fisher-Yatesシャッフル状態管理パターンsetTimeoutイベントリスナー
Step 4

クリア判定と演出

全ペアを見つけたらゲームクリア。完了の検出と結果表示を実装。

  • 全ペア完了の検出
  • クリアメッセージの表示
ここで学ぶこと: ゲーム完了条件UIフィードバック
