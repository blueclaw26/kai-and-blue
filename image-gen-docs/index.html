画像生成ツール

AI画像生成ツールの制作記録

制作ステップ

Step 1

画像生成APIの選定

Pollinations AI（無料、キー不要）を選定。APIキーなしで画像生成が可能なサービスを比較検討し、手軽さと品質のバランスからPollinationsを採用。

ここで学ぶこと: 画像生成AIPollinations AIAPIエンドポイント
Step 2

プロンプト設計

テキストからの画像生成、スタイルプリセット、ネガティブプロンプトの仕組みを実装。ユーザーの入力にスタイル指定や除外条件を自動付加する設計。

ここで学ぶこと: プロンプトエンジニアリングネガティブプロンプトスタイルプリセット
Step 3

UIの実装

入力フォーム、ローディング状態、画像表示、ダウンロード機能を実装。APIのレスポンスがimg要素のsrcとして直接使える特性を活用したシンプルな設計。

ここで学ぶこと: img要素のsrc動的変更ローディングUIBlob/URLダウンロード
Step 4

ユーザー体験の改善

履歴保存、ランダムプロンプト、シード固定による再現性の確保。過去のプロンプトをクリックで再利用できるチップUIと、創造性を刺激するランダム機能。

ここで学ぶこと: localStorage履歴シード値再現性

Kaiのレビュー

  • まず無料APIで動くUIを確認し、品質に応じてAPI差し替え可能な設計に
  • プロンプトの組み立てロジックをbuildPrompt()関数に集約して差し替えやすくした
  • スタイルプリセットでAI知識がなくても使いやすいUIに

使用技術

  • HTML
  • CSS
  • JavaScript
  • Pollinations AI API
  • localStorage
