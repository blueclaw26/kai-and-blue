Web & AIクイズの制作記録

概要

「AIと開発する人間が知っておくべき20のこと」をテーマにした4択クイズ。

制作ステップ

Step 1

問題データの設計

20問の問題を配列オブジェクトとして定義。コードの書き方ではなく、概念理解にフォーカス。

  • 各問題: 質問文、4つの選択肢、正解インデックス、解説テキスト
  • テーマ: Web技術、AI概念、インターネットの仕組み
ここで学ぶこと: データ構造の設計、配列とオブジェクトの使い分け
Step 2

1問ずつ表示するUI

現在の問題を画面に描画し、進捗を視覚的に表示する。

  • 4つのボタンを動的に生成
  • プログレスバーで進捗を表示（current / total）
ここで学ぶこと: 動的なDOM生成、プログレスバーの実装
Step 3

正誤判定とフィードバック

正解・不正解に応じて、異なるフィードバックを表示する。

  • 正解→ボタン緑＋「✓」＋紙吹雪（canvas-confetti）＋「正解！」テキスト→1秒で自動遷移
  • 不正解→ボタン赤＋「✗」＋正解を緑で表示＋解説テキスト＋「次へ」ボタン
ここで学ぶこと: 条件分岐、外部ライブラリ（CDN）の利用、UXの設計
Step 4

結果画面

全問回答後にスコアと正答率を表示し、リプレイを促す。

  • スコアと正答率を表示
  • 正答率に応じたメッセージ（100%→「完璧」等）
  • リスタートボタン
ここで学ぶこと: スコアリングロジック、状態のリセット

Kaiのレビューで変わったこと

  • 正解時の「✓」だけでは確認しづらい→紙吹雪エフェクト追加
  • 不正解時の自動遷移→「次へ」ボタンに変更（解説を読む時間確保）
  • 間違えた時に解説を表示する機能追加（「ただのチェックじゃなく学べるように」）
  • テーマを「プログラミング」から「Web & AI概念」に変更（「コードはAIが書くから」）

使用技術まとめ

  • HTML
  • CSS
  • JavaScript
  • canvas-confetti (CDN)
  • データ構造設計
