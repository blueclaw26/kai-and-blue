Markdownプレビューア

リアルタイムMarkdownエディタの制作記録

概要

左にMarkdownを書くと右にHTMLプレビューがリアルタイムで表示されるエディタ。外部ライブラリを使わず、正規表現で自作パーサーを実装。

制作ステップ

Step 1

分割レイアウトの構築

左右50/50の分割ペインを作り、エディタとプレビューを横に並べる。

  • CSS Flexboxで横並び、モバイルでは縦積み
  • エディタ側はtextarea、プレビュー側はdiv
ここで学ぶこと: Flexbox textarea
Step 2

Markdownパーサーの実装

正規表現でMarkdown記法をHTMLタグに変換する。ブロック要素を先に、インライン要素を後に処理する順序が重要。

  • 見出し（#）、太字（**）、斜体（*）、コード（`）、リンク、画像、リスト、引用、水平線
  • 段落とコードブロックの処理（ブロック要素は先に処理、インライン要素は後）
ここで学ぶこと: 正規表現 パーサー ブロック要素とインライン要素
Step 3

リアルタイムプレビュー

textareaのinputイベントを監視し、キー入力のたびにパーサーを実行してプレビューを更新する。

  • キー入力のたびにパーサーを実行し、プレビューのinnerHTMLを更新
  • デフォルトコンテンツとしてMarkdownチートシートを表示
ここで学ぶこと: inputイベント innerHTML
Step 4

ユーティリティ機能

エディタの使い勝手を向上させるための便利機能を追加する。

  • HTMLソースのクリップボードコピー（navigator.clipboard API）
  • エディタのクリア
  • Tabキーでスペース挿入（デフォルトのフォーカス移動を防止）
ここで学ぶこと: Clipboard API preventDefault

Kaiのレビューで変わったこと

  • サブエージェントで一発作成、大きな修正なし
